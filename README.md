# Controle-de-Gastos-WEB
Projeto de controle de gastos, criado para te ajudar no seu dia a dia.
Desenvolvi meu primeiro sistema web com Python e Flask!

Como estudante de Análise e Desenvolvimento de Sistemas, decidi criar um projeto prático para aplicar tudo que venho aprendendo em programação.

💻 Desenvolvi um Sistema de Controle de Gastos Pessoais, onde é possível:

✔️ Adicionar gastos
✔️ Editar e excluir registros
✔️ Visualizar resumo financeiro
✔️ Organizar despesas de forma simples e funcional

🛠️ Tecnologias utilizadas:
 • Python
 • Flask
 • HTML
 • CSS
 • Lógica de Programação

📚 Durante o desenvolvimento, aprendi na prática:
 • Estrutura de um projeto web
 • Conceito de CRUD (Create, Read, Update, Delete)
 • Rotas no Flask
 • Integração entre back-end e front-end

Esse projeto representa mais um passo na minha evolução como desenvolvedor.
Estou focado em evoluir diariamente e construir projetos cada vez mais completos.
